# Sistema de Facturación SaaS

Plataforma de facturación multi-tenant para la República Dominicana.

## Estructura del proyecto

- `backend/` — API REST con Node.js/Express + PostgreSQL
- `web/` — App web con React
- `mobile/` — App móvil con React Native

## Requisitos
- Node.js 18+
- PostgreSQL
- npm

## Inicio rápido
cd backend
npm install
npm run dev
